minimal requirements
  RAM : 2GB
  OS  : Linux
  CPU : Intel Atom N450 with 1,66 GHz

Maptool controls:
* CTRL + R = Reload the UI (you need this)
* Arrow Up or Down = Change Edit layer
* WASD = Move to the next map
